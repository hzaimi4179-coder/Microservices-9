 #   Microservices-9 : Client React pour API REST

## Installation de Node.js, npm et Create React App

### 1. Installer Node.js

Télécharger la version LTS depuis : https://nodejs.org/

### 2. Vérifier l'installation :

``` bash
node -v
npm -v
```

### 3. Installer Create React App :

``` bash
npm install -g create-react-app
```

## Création du Projet React

``` bash
npx create-react-app compte-client
cd compte-client
```

### Installer Bootstrap :

``` bash
npm install bootstrap
```

Ajouter dans `src/index.js` :

``` javascript
import 'bootstrap/dist/css/bootstrap.min.css';
```

``` bash
npm start
```

 http://localhost:3000/
